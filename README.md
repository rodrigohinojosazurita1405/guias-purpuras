# 🟣 Guías Púrpuras

Plataforma de clasificados para Bolivia - Encuentra profesionales, restaurantes, trabajos y servicios.

## 🚀 Tecnologías

- **Frontend**: Vue 3 + Vuestic UI
- **Backend**: Django + PostgreSQL (En desarrollo)

## 📦 Instalación
```bash
cd frontend
npm install
npm run dev
```

## 🎨 Paleta de Colores

- Púrpura Principal: `#5C0099`
- Amarillo: `#FDC500`
